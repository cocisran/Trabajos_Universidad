*******Como jugar este tetris:

Para poder jugar este tetris de momento tienes que encontrarte en linux y contar con una instalacion de java 11 o superior
la herramienta  que usamos para compilar el proyecto se llama Gradle y te recomendamos que tambien la uses por que es simple
y sencilla de usar.

Si no cuentas con gradle puedes ejecutar los siguientes comandos en la terminal : 

    1) $ sdk install gradle 6.0.1

Y en caso de no tener sdkman instaldo tambien realiza estos pasos : 
En una terminal :

    1) $ curl -s "https://get.sdkman.io" | bash
    2) $ source "$HOME/.sdkman/bin/sdkman-init.sh"
    3) $ sdk version

Este ultimo, para verificar que la instalacion haya sido correcta

Y si eres de aquellos que no confian en nadie no te preocupes siente libre de seguir los pasos por tu cuenta desde las 
fuentes oficiales :

    para gradle https://gradle.org/install/
    para sdkman https://sdkman.io/install

Una vez hecho esto tan solo ejecuta este ultimo comando abriendo una terminal en el directorio tetris : 

    1) $ gralde run 

¡Y listo! dentro de unos segundos se desplegará una ventana donde comenzará el juego  :D

                            ************REGLAS DEL JUEGO************

ATENTO: la musica suena y las piezas comienzan a caer,  ¿Que tan bien podras acomodar el tetronimo quede bloqueado?

CONTROLES

    UP    ARROW     Rota el tetronimo a la derecha      
    DOWN  ARROW     Hace caer el tetronimo mas rápido             (caida suave)
    RIGHT ARROW     Mueve el tetronimo a la derecha
    LEFT  ARROW     Mueve el tetronimo a la izquierda

    TECLA     C     Guarda el tetronimo que cae y lo intercambia por el siguiente o el guardado
    TECLA     Z     Rota el tetronimo a la izquierda
    TECLA SPACE     CUIDADO hace caer el tetronimo hasta el piso  (caida violenta)

NIVELES

    Entre mas tiempo juegues te daras cuenta que las piezas caen cada ves mas rapido, matente siempre preparado :D 
    No te preocupes a mayor dificultad mayor recompensa, las piezas caen mas rapido y tu puntuacion subira mas rapido
    tambien

PUNTOS
    Borrar
    1 linea     100 puntos
    2 lineas    300 puntos
    3 lineas    500 puntos
    4 lineas    800 puntos

    caida suave     1 punto   x  Bloque caido
    caida violenta  2 puntos  x  Bloque caido

Los puntos se multiplican por el nivel actual :p
